# Automative-Car-Dashboard-System-CAN
A distributed embedded systems project implementing a multi-node automotive dashboard using the Controller Area Network (CAN) protocol on PIC18F4580 microcontrollers. The system simulates inter-ECU communication in vehicular environments, enabling real-time data exchange between sensor nodes and a central dashboard unit.

# Features
- Implements CAN-based communication between multiple ECUs
- ECU1 and ECU2 act as independent sensor nodes
- ECU3 functions as the central dashboard controller
- Captures vehicle parameters using ADC interfacing
- Displays live vehicular data on Character LCD (CLCD)

# Technical Skills
- Embedded C programming on PIC18F4580
- CAN protocol implementation for inter-ECU communication
- ADC configuration and analog signal acquisition
- CLCD interfacing and display control logic
